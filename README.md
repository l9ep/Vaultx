# yuki's space

A personal website and script drop zone by **yuki** — a Roblox coder who posts scripts, exploits, and Lua code for anyone to use for free.

## What is this?

This is yuki's personal site where he drops his Roblox scripts and code files. Anyone who visits can browse, copy, download, or view the raw source of any uploaded file — no sign up, no cost, just grab what you need.

## What you can do

- **Browse scripts** — view all uploaded Roblox scripts and files
- **Copy code** — one click copy any script to your clipboard
- **Download files** — download any file directly to your device
- **Raw view** — open a dedicated raw page for any file with line numbers and a clean copy button
- **Search & filter** — search files by name or content, filter by type (`.lua`, `.js`, `.txt`, etc.)
